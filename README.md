# Task 2: Customer Segmentation

## Objective
The objective of this task is to segment customers based on their purchasing behavior using RFM (Recency, Frequency, Monetary) analysis.

## Dataset
Online Retail Dataset (Kaggle)

Key Features:
- Invoice Date
- Customer ID
- Quantity
- Unit Price

## Methodology
1. Loaded and cleaned the dataset.
2. Removed missing customer IDs and invalid transactions.
3. Calculated RFM metrics:
   - Recency
   - Frequency
   - Monetary Value
4. Standardized the data.
5. Applied K-Means clustering to segment customers.

## Models Used
- K-Means Clustering

## Results
Customers were grouped into multiple segments based on purchasing behavior, such as:
- High-value loyal customers
- Regular customers
- Low-engagement customers

## Conclusion
Customer segmentation helps businesses design targeted marketing strategies and improve customer retention.
